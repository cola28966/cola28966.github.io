Redis (四) 有序集合

有序集合简介

  • 有序集合保留了集合元素不能重复的特点

  • 有序集合会给每个元素设置一个分数，并以此作为排序的依据

  • 有序集合不能包含相同的元素，但是不同元素的分数可以相同

    基本命令

  • 添加元素

    nx 表示存在时才添加成功

  • 查看范围内的元素

  • 正序输出

  • 逆序输出

  • 查看元素的排名

    查看 bbb 在 sores 的排名，返回的值为排名

  • 统计所有元素的数量

  • 统计区间内元素的数量

  • 删除元素

  • 按最小弹出元素

  • 阻塞弹出

典型使用场景

  • 排行榜

    数据示例 key -> (加权分：帖子ID，加权分：帖子ID，加权分：帖子ID，…)

    使用命令: zrevrange key 0 -1

  • 延续队列

    数据示例 key -> (时间戳：用户ID，时间戳：用户ID，时间戳：用户ID，…)

    使用命令：zrange key 0 0 , zremrangebyrank key 0 0 / zpopmin

    应用程序：轮询集合中的数据(不断的查看第一个数据)，根据分数判断该数据是否超时，若超时则做出处理并移除数据
